# React Developer - ΕΚΠΑ E-Learning

Ασκήσεις από το **Πρόγραμμα Επαγγελματικής Επιμόρφωσης και Κατάρτισης** στη σύγχρονη ανάπτυξη ιστοσελίδων, από το πρόγραμμα [E-Learning του Εθνικού και Καποδιστριακού Πανεπιστημίου Αθηνών (ΕΚΠΑ)](https://www.uoa.gr/).

## Τεχνολογίες

React 18 &bull; TypeScript &bull; Vite &bull; React Bootstrap &bull; Bootstrap 5

## Επισκόπηση Μαθημάτων

Το πρόγραμμα περιλαμβάνει 7 θεματικές ενότητες. Οι παρακάτω ασκήσεις αφορούν τις ενότητες React:

| # | Μάθημα | Θεματολογία |
|---|--------|-------------|
| 1 | UX-UI Design | Αρχές σχεδιασμού UX/UI & πρακτική |
| 2 | HTML5 & CSS | Responsive σελίδες, Flexbox, Grid, templates |
| 3 | Bootstrap 5 | Components, layout, πρακτική |
| 4 | JavaScript & ES14 | ES2023, Node.js, testing |
| **5** | **React Basic** | **Components, state, props, styling** |
| **6** | **React Advanced** | **Hooks, forms, data fetching, testing** |
| **7** | **React Practice** | **Ολοκληρωμένες εφαρμογές** |

## Ασκήσεις

### React Basic

| Άσκηση                                                       | Περιγραφή | Βασικές Έννοιες |
|--------------------------------------------------------------|-----------|-----------------|
| [01-counter](05-react-basic/01-counter/)                     | Εμφάνιση ονόματος σε h1 και button counter που διπλασιάζεται (10, 20, 40, 80...) | Components, useState, event handling |
| [02-components](05-react-basic/02-components/)               | Τρία custom components, το καθένα με διαφορετικό background color | Component composition, props |
